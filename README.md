# Pyfuck
A modern dialect of the best programming language ever.

[Hello, World!](examples/hello.py)
## Syntax
Pyfuck | Brainfuck
-------|----------
`raise` | `+` 
`break` | `-` 
`pass` | `>`
`return` | `<` 
`print()` | `.` 
`while True:` | `[`
unindent | `]`


## Build
```
stack install
```

## Run
```
pyfuck <source-path>
```
